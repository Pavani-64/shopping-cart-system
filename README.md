# 🛍️ Smart Shopping Cart System

## 🚀 Features

* 🔐 User Login & Signup
* 🛒 Add to Cart
* 🗑️ Remove Items
* 💳 Checkout System
* 🗄️ SQLite Database
* 🎨 Responsive UI (Bootstrap)

## 🛠️ Tech Stack

* Python
* Flask
* SQLite
* HTML/CSS (Bootstrap)

## ▶️ How to Run

```bash
pip install flask
python app.py
```
